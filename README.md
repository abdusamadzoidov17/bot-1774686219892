# Telegram Support Bot

Бот для службы поддержки на grammY + TypeScript для Vercel (Next.js App Router).

## Функции

- Пользователи:
  - Создание обращений (/support или кнопка)
  - Просмотр своих тикетов (в разработке)
  - Отмена тикета

- Администраторы:
  - Панель управления (/admin)
  - Просмотр всех тикетов
  - Ответы на обращения
  - Закрытие тикетов

## Установка

1. Клонируйте проект
2. Установите зависимости: `npm install`
3. Создайте бота через @BotFather и получите токен
4. Настройте переменные окружения в Vercel:
   - `BOT_TOKEN` (обязательно)
   - `ADMIN_IDS` (список ID админов через запятую)
5. Разверните на Vercel

## Примечания

- Для продакшена замените заглушки в `src/middleware/storage.ts` на реальную БД (Drizzle + Vercel Postgres) или KV
- Добавьте обработку медиа-файлов (фото, документы) в `messages.ts`
- Реализуйте очередь уведомлений для админов
- Добавьте аутентификацию админов через пароль или 2FA
